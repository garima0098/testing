JavaScript

The programming language of the web adding interactivity, logic, and dynamic behaviour.

What is JavaScript?

JavaScript (JS) is a lightweight, interpreted programming language that runs in the browser. It allows developers to create dynamically updating content, control multimedia, animate images, and respond to user actions all without reloading the page.

JavaScript is also used server-side with Node.js, making it a true full stack language.

Key JavaScript Concepts

  • Variables — var, let, const
  • Data Types — String, Number, Boolean, Array, Object, null, undefined
  • Functions — declarations, expressions, arrow functions
  • DOM Manipulation — getElementById, querySelector, innerHTML
  • Events — addEventListener, click, input, submit
  • Conditionals — if/else, ternary operator, switch
  • Loops — for, while, forEach, map, filter
  • Async JS — Promises, async/await, fetch API
  • ES6+ Features — template literals, destructuring, spread operator

JavaScript Code Examples

//  Variables 
const name  = "Garima";
let   score = 95;

//  Arrow Function 
const greet = (name) => {
  return `Hello, ${name}!`;
};
console.log(greet("Garima"));
// → "Hello, Garima!"

// DOM Manipulation 
const btn = document.querySelector('#myBtn');
const box = document.querySelector('#output');

btn.addEventListener('click', () => {
  box.textContent = "Button clicked!";
  box.style.color = "green";
});

//  Array Methods 
const scores = [45, 72, 88, 91, 60];
const passed = scores.filter(s => s >= 60);
const doubled = scores.map(s => s * 2);
const total  = scores.reduce((a,b) => a + b, 0);

//  Async / Fetch API 
async function getUser(id) {
  try {
    const res  = await fetch(
      `https://api.example.com/users/${id}`
    );
    const data = await res.json();
    console.log(data.name);
  } catch (err) {
    console.error("Error:", err);
  }
}

//  Destructuring 
const { firstName, age } = userObject;
const [first, ...rest]   = myArray;

//  Template Literals 
const msg = `Score: ${score}/100`;

//  Ternary Operator 
const grade = score >= 60 ? "Pass" : "Fail";
JavaScript Tip
Always use const by default. Only use let when you need to reassign a variable. Avoid var in modern JavaScript it has confusing scope rules.
