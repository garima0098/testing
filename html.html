HTML

HyperText Markup Language the structure and skeleton of every web page.

What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create the structure of web pages. It uses elements (tags) to describe different types of content such as headings, paragraphs, images, and links.

Every web page you visit is built on an HTML foundation. Browsers read HTML documents and render them as visible pages.

Key HTML Concepts

  • Semantic Elements — <header>, <main>, <footer>, <article>, <section>
  • Forms & Inputs — <form>, <input>, <textarea>, <select>
  • Media — <img>, <video>, <audio>
  • Links & Navigation — <a href="">, <nav>
  • Tables — <table>, <thead>, <tbody>, <tr>, <td>
  • Meta Tags — for SEO, character encoding, and viewport settings
  • Accessibility — alt, aria-label, role attributes

Common HTML Tags

TagPurposeExample
<h1>–<h6>Headings<h1>Title</h1>
<p>Paragraph<p>Text</p>
<a>Hyperlink<a href="url">
<img>Image<img src="img.jpg">
<div>Block container<div class="box">
<span>Inline container<span>word</span>
<ul> / <ol>Lists<li>item</li>
<form>Form container<form method="POST">
<input>Input field<input type="text">

Full HTML Page Example

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width, initial-scale=1.0">
    <meta name="description"
      content="My portfolio page">
    <title>Garima's Portfolio</title>
    <link rel="stylesheet" href="style.css">
  </head>
  <body>
    <header>
      <nav>
        <a href="index.html">Home</a>
        <a href="about.html">About</a>
      </nav>
    </header>
    <main>
      <section id="hero">
        <h1>Welcome to My Portfolio</h1>
        <p>I am a full stack developer.</p>
        <img src="profile.jpg" alt="Garima">
      </section>
      <section id="skills">
        <h2>My Skills</h2>
        <ul>
          <li>HTML5</li>
          <li>CSS3</li>
          <li>JavaScript</li>
        </ul>
      </section>
      <section id="contact">
        <h2>Contact Me</h2>
        <form action="submit.php" method="POST">
          <input type="text" name="name"
            placeholder="Your name" required>
          <input type="email" name="email"
            placeholder="Your email" required>
          <textarea name="message"></textarea>
          <button type="submit">Send</button>
        </form>
      </section>
    </main>
    <footer>
      <p>&copy; 2025 Garima Adhikari</p>
    </footer>
  </body>
</html>
Why Semantic HTML?
Using semantic tags like <header>, <main>, and <footer> instead of generic <div> tags improves accessibility for screen readers and helps search engines understand the page structure (SEO).
