Bootstrap Framework

The world's most popular CSS framework for building responsive, mobile first websites.

What is Bootstrap?

Bootstrap is a free, open source CSS framework developed by Twitter. It provides pre built CSS classes, a responsive grid system, and JavaScript components that help developers build professional websites quickly without writing everything from scratch.

Key Bootstrap Features

  • 12-Column Grid System — responsive layout using col-* classes
  • Breakpoints — xs, sm, md, lg, xl, xxl
  • Utility Classes — spacing, colour, display, flex, text alignment
  • Components — navbar, cards, modals, carousels, accordions
  • Forms — styled inputs, validation, select dropdowns
  • Icons — Bootstrap Icons library (1,800+ icons)
  • JavaScript Plugins — modals, tooltips, popovers

Accordion

The Bootstrap grid system uses containers, rows, and columns to layout content. It's built with Flexbox and is fully responsive.

Mobile first means styles are written for small screens first, then enhanced for larger screens using media queries. Bootstrap's grid classes like col-sm-6 follow this approach.

Bootstrap utility classes let you apply styling directly in HTML without writing custom CSS. Examples: mt-3 (margin-top), text-center, d-flex, bg-primary, fw-bold.
