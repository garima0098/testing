CSS

Cascading Style Sheets making web pages visually appealing and responsive.

What is CSS?

CSS (Cascading Style Sheets) is the language used to style and layout web pages. While HTML provides structure, CSS controls colours, fonts, spacing, positioning, animations, and responsive design.

Key CSS Concepts

  • Selectors — element, class (.name), ID (#name), pseudo class (:hover)
  • Box Model — margin, border, padding, content
  • Flexbox — one-dimensional layout system
  • CSS Grid — two-dimensional layout system
  • Media Queries — responsive breakpoints
  • CSS Variables — reusable custom properties
  • Transitions & Animations — smooth visual effects
  • Pseudo elements — ::before, ::after

CSS Code Example

/*  CSS Variables  */
:root {
  --primary: #0d6efd;
  --danger:  #dc3545;
  --radius:  8px;
}

/*  Selectors  */
h1    { color: var(--primary); }
.card { border-radius: var(--radius); }
#hero { background: #f8f9fa; }

/*  Box Model  */
.box {
  margin: 10px;
  padding: 20px;
  border: 1px solid #ccc;
  border-radius: var(--radius);
}

/*  Flexbox  */
.navbar {
  display: flex;
  align-items: center;
  justify-content: space-between;
  gap: 16px;
}

/*  CSS Grid  */
.grid-layout {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

/*  Transitions  */
.btn {
  transition: background 0.3s, transform 0.2s;
}
.btn:hover {
  background: var(--danger);
  transform: scale(1.05);
}

/*  Media Query  */
@media (max-width: 768px) {
  .grid-layout {
    grid-template-columns: 1fr;
  }
}

/*  Keyframe Animation  */
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(20px); }
  to   { opacity: 1; transform: translateY(0); }
}
.hero { animation: fadeIn 0.8s ease; }
CSS Tip
Use CSS custom properties (--variables) to keep your colours and values consistent across your stylesheet. Changing one variable updates the entire site.
